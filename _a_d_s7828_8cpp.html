Detailed Description

ADS7828 chip's driver for Arduino.

Functions for using the 12-Bit, 8-Channel Sampling ANALOG-TO-DIGITAL CONVERTER with I2C Interface ADS7828.

For more informations, refer to the ADS7828's data sheet.
