Enumeration Type Documentation

enum eProtocolFrame

Enumeration of all frame's type.

This enumeration allow to map frame's type to an ID. Values are generated at compilation time according to protocol_anifests.h . Each member respect the pattern cProtocolFrameIDF where IDF is the ID of the considered frame.

Note
this enumeration should be used as an index for array who groups data from frames.
